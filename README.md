# Weblooks — Site Institucional

Site institucional moderno e responsivo para a **Weblooks**, empresa de desenvolvimento web e soluções digitais liderada por **Berg Pinheiro**.

## 🚀 Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS
- **Animações:** Framer Motion
- **Ícones:** Lucide React
- **Deploy:** Vercel

## 📦 Instalação

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.

## 🏗️ Estrutura

```
src/
├── app/                    # App Router (Next.js)
│   ├── page.tsx            # Home page
│   ├── sobre/page.tsx      # Sobre page
│   ├── servicos/page.tsx   # Serviços page
│   └── contato/page.tsx    # Contato page
├── components/
│   ├── layout/             # Header, Footer
│   ├── sections/           # Seções reutilizáveis
│   └── ui/                 # WhatsApp button, etc
├── lib/
│   ├── constants.ts        # Dados centralizados
│   └── utils.ts            # Utilitários
└── hooks/                  # Custom hooks
```

## 🌐 Deploy

Pronto para deploy na **Vercel**:

```bash
npm run build
```
